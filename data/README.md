\# Data Directory



This directory holds locally cached data files for development.

These files are not tracked in git.



\## rounds.parquet

The processed disc golf rounds dataset, generated by the refresh Lambda.

To generate locally, run the refresh Lambda function directly:



```bash

cd backend/refresh

python lambda\_function.py

```



\## Data Sources

\- Round scores: Google Sheets (publicly accessible CSV export)

\- Weather data: Open-Meteo historical weather API (free, no API key required)
